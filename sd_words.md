```
Stable Diffusion과 관련된 주요 용어들은 다음과 같습니다:

Diffusion Model: 확산 모델은 데이터의 분포를 학습하기 위해 노이즈를 점진적으로 제거하는 방식으로 작동하는 생성 모델입니다.

Latent Space: 데이터의 저차원 표현 공간으로, 원본 데이터의 중요한 특징을 압축하여 표현합니다.

Noise Schedule: 확산 과정에서 노이즈를 추가하거나 제거하는 일정표입니다. 이는 모델이 데이터를 점진적으로 복원하는 데 중요한 역할을 합니다.

Denoising: 노이즈가 추가된 데이터를 원래 상태로 복원하는 과정입니다.

Checkpoint: 모델의 특정 상태를 저장한 파일로, 훈련 중간에 모델의 가중치와 옵티마이저 상태 등을 저장합니다.

Sampling: 학습된 모델을 사용하여 새로운 데이터를 생성하는 과정입니다.

Encoder: 데이터를 저차원 잠재 공간으로 변환하는 신경망입니다.

Decoder: 잠재 공간의 표현을 원래 데이터 공간으로 복원하는 신경망입니다.

Latent Variable: 관측되지 않은 변수로, 데이터의 잠재적 특성을 나타냅니다.

Variational Inference: 잠재 변수의 분포를 근사하기 위해 사용하는 방법입니다.

ELBO (Evidence Lower Bound): 변분 추론에서 사용되는 목적 함수로, 모델의 학습을 최적화하기 위해 사용됩니다.

Annealing: 학습 과정에서 노이즈의 크기를 점진적으로 줄이는 방법입니다.

Reverse Process: 확산 모델에서 노이즈를 제거하여 원래 데이터를 복원하는 과정입니다.

Forward Process: 원본 데이터에 점진적으로 노이즈를 추가하는 과정입니다.

Score Matching: 데이터의 확률 밀도 함수의 기울기를 학습하는 방법입니다.

Likelihood: 주어진 데이터가 모델에 의해 생성될 확률입니다.

Prior Distribution: 잠재 변수의 사전 분포로, 모델이 학습하기 전에 설정됩니다.

Posterior Distribution: 주어진 데이터에 대한 잠재 변수의 사후 분포입니다.

Reparameterization Trick: 변분 오토인코더에서 잠재 변수의 샘플링을 가능하게 하는 방법입니다.

Generative Model: 새로운 데이터를 생성할 수 있는 모델입니다.
```

```
IT 비전공자인 디자이너가 WEB GUI를 통해 Stable Diffusion으로 이미지를 생성할 때 필요한 최소한의 용어는 다음과 같습니다:

Stable Diffusion: 이미지를 생성하는 데 사용되는 모델의 이름입니다.
Checkpoint: 모델의 특정 상태를 저장한 파일로, 이미지를 생성할 때 불러옵니다.
Sampling: 모델을 사용하여 새로운 이미지를 생성하는 과정입니다.
Prompt: 이미지를 생성하기 위해 입력하는 텍스트 설명입니다.
Resolution: 생성할 이미지의 해상도입니다.
Iterations/Steps: 이미지를 생성하는 데 필요한 반복 횟수입니다.
Seed: 이미지 생성의 초기값으로, 동일한 시드를 사용하면 동일한 이미지를 생성할 수 있습니다.
Output: 생성된 최종 이미지입니다.
```

```
디자이너가 이미지를 조금씩 변경하면서 생성할 때, 커스텀할 수 있는 부분과 관련된 용어들은 다음과 같습니다:

Prompt: 이미지를 생성하기 위해 입력하는 텍스트 설명입니다. 프롬프트를 변경하여 다양한 이미지를 생성할 수 있습니다.

Seed: 이미지 생성의 초기값으로, 동일한 시드를 사용하면 동일한 이미지를 생성할 수 있습니다. 시드를 변경하여 다양한 변형을 시도할 수 있습니다.

Iterations/Steps: 이미지를 생성하는 데 필요한 반복 횟수입니다. 더 많은 반복 횟수를 설정하면 이미지의 품질이 향상될 수 있습니다.

Guidance Scale: 텍스트 프롬프트와 이미지 간의 일치도를 조절하는 파라미터입니다. 높은 값은 프롬프트에 더 충실한 이미지를 생성하고, 낮은 값은 더 창의적인 이미지를 생성할 수 있습니다.

Resolution: 생성할 이미지의 해상도입니다. 해상도를 변경하여 이미지의 크기와 세부 사항을 조절할 수 있습니다.

Strength: 원본 이미지와 생성된 이미지 간의 유사도를 조절하는 파라미터입니다. 이 값을 조절하여 원본 이미지에서 얼마나 벗어날지를 결정할 수 있습니다.

Noise Level: 이미지 생성 과정에서 추가되는 노이즈의 양입니다. 노이즈 레벨을 조절하여 이미지의 다양성을 높일 수 있습니다.

Latent Space: 데이터의 저차원 표현 공간으로, 원본 데이터의 중요한 특징을 압축하여 표현합니다. 잠재 공간의 특정 부분을 조작하여 이미지를 변경할 수 있습니다.

Interpolation: 두 개 이상의 잠재 벡터 사이를 보간하여 새로운 이미지를 생성하는 방법입니다. 이를 통해 두 이미지 간의 중간 형태를 생성할 수 있습니다.

Inpainting: 이미지의 특정 부분을 수정하거나 채우는 기술입니다. 이미지를 부분적으로 변경할 때 유용합니다.

Outpainting: 기존 이미지의 경계를 넘어 새로운 부분을 생성하는 기술입니다. 이미지를 확장할 때 사용됩니다.

이 용어들은 디자이너가 이미지를 커스텀하면서 생성할 때 이해하고 있어야 하는 기본적인 개념들입니다. 이를 통해 디자이너는 원하는 이미지를 더 세밀하게 조정하고 다양한 변형을 시도할 수 있습니다.
```
